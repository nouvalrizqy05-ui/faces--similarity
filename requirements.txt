## рҹ“Ӣ Deskripsi

Proyek ini mengimplementasikan sistem deteksi kemiripan wajah menggunakan konsep matematika dari mata kuliah **Aljabar Linear**:

| Konsep | Implementasi |
|---|---|
| **Eigenvalue** | Mengukur besar informasi per principal component |
| **Eigenvector** | Arah principal component = *eigenfaces* |
| **SVD** | Dekomposisi matriks data untuk mencari eigenfaces |
| **PCA** | Reduksi dimensi wajah dari 10.000 вҶ’ 50 fitur |
| **Cosine Similarity** | Mengukur kemiripan antara dua wajah di ruang PCA |

**Dataset**: [FG-NET Face and Gesture Recognition Network](https://yanweifu.github.io/FG_NET_data/) вҖ” 1.002 foto wajah dari 82 orang dalam rentang usia yang luas.

## рҹ”ў Alur Matematika

### 1. Load & Preprocessing
```
Setiap gambar (100Г—100 px) вҶ’ grayscale вҶ’ normalisasi [0,1] вҶ’ flatten вҶ’ vektor (10.000,)
```

### 2. Centering
$$X_c = X - \mu \quad \text{(setiap gambar dikurangi mean face)}$$

### 3. SVD (inti PCA)
$$X_c = U \Sigma V^T$$

- **U** вҲҲ в„қ^(mГ—m): left singular vectors
- **ОЈ** вҲҲ в„қ^(mГ—n): singular values
- **V^T** вҲҲ в„қ^(nГ—n): right singular vectors вҶ’ **eigenfaces**

### 4. Eigenvalue
$$\lambda_i = \frac{\sigma_i^2}{m-1}$$

### 5. Pilih k Komponen (Explained Variance)
$$\text{EVR}_k = \frac{\sum_{i=1}^{k}\lambda_i}{\sum_{i=1}^{p}\lambda_i} \geq 95\%$$

### 6. Proyeksi
$$Z = X_c \cdot V_k^T \quad \text{(10.000 dimensi вҶ’ k dimensi)}$$

### 7. Rekonstruksi
$$\hat{X} = Z \cdot V_k + \mu$$

### 8. Kemiripan Wajah
$$\text{sim}(z_1, z_2) = \frac{z_1 \cdot z_2}{\|z_1\| \|z_2\|} \geq \text{threshold} \implies \text{MIRIP}$$

## рҹҡҖ Cara Menjalankan

### A. Clone & Setup

```bash
git clone https://github.com/YOUR_USERNAME/face-similarity-pca.git
cd face-similarity-pca
pip install -r requirements.txt
```

### B. Siapkan Dataset FG-NET

Download dataset dari [FG-NET](https://yanweifu.github.io/FG_NET_data/), lalu:

```bash
# Reorganisasi dari flat ke per-orang (otomatis)
python src/reorganize_dataset.py --input images_raw/images/ --output dataset/
```

### C. Training PCA

```bash
python src/train.py --dataset dataset/ --n_components 50 --output models/
```

Output:
```
[1/5] Loading dataset: 1002 gambar dari 82 orang
[2/5] SVD selesai (5.4s)
[3/5] 90% variance вҶ’ 50 komponen
[4/5] MSE: 0.005234 | PSNR: 22.81 dB | SSIM: 0.9205 | CR: 17.89x
[5/5] Model disimpan ke models/
```

### D. Bandingkan Dua Wajah

```bash
# Compare dua gambar
python src/compare.py dataset/001/001A02.JPG dataset/001/001A18.JPG

# Identifikasi dari database
python src/compare.py dataset/002/002A05.JPG --mode recognize
```

### E. Jalankan Streamlit App (lokal)

```bash
streamlit run main.py
```

Buka: `http://localhost:8501`

### F. Jalankan Notebook (Colab)

1. Buka [Google Colab](https://colab.research.google.com)
2. Upload `notebooks/Face_Similarity_PCA.ipynb`
3. Upload `images.zip` (dataset FG-NET)
4. Jalankan semua cell
